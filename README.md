# UPT Home Page

Homepage oficial de la **Universidad Privada de Tacna (UPT)**.

## 🌐 Sitio Web

🔗 [https://nkmelndz.github.io/upt-home-page/](https://nkmelndz.github.io/upt-home-page/)

## Descripción

Este proyecto es la página de inicio (homepage) de la Universidad Privada de Tacna. Incluye secciones de navegación para postulantes, estudiantes, egresados, empleabilidad y más.

## Despliegue

El sitio se despliega automáticamente en GitHub Pages mediante GitHub Actions al hacer push a la rama `main`.
