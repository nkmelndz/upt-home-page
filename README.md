# 🎓 UPT Home Page

Alternativa a la página de inicio de la **Universidad Privada de Tacna (UPT)**, desarrollada como proyecto del curso de **Diseño de Interfaces**. Construida con HTML, CSS y JavaScript.

🔗 **Sitio en vivo:** [https://nkmelndz.github.io/upt-home-page/](https://nkmelndz.github.io/upt-home-page/)

## 🎯 Propósito

Este proyecto es una **alternativa** a la homepage oficial de la Universidad Privada de Tacna, creada con fines académicos en el marco del curso de Diseño de Interfaces. Sirve como ejercicio de maquetación web y está desplegado en GitHub Pages para su visualización pública.

> ⚠️ Este sitio **no es una réplica oficial** de la UPT ni está afiliado a la institución.

## 🛠️ Tecnologías

* 🌐 HTML5
* 🎨 CSS3
* ⚡ JavaScript / [jQuery 3.6](https://jquery.com/)
* 🚀 GitHub Actions (despliegue automático en GitHub Pages)

## 📄 Secciones de la página

| Sección | Contenido |
|---|---|
| 📝 **Postulantes** | Modalidades de admisión, categorías de pago, simulador de pensiones, inscripción en línea |
| 🎒 **Estudiantes** | Matrícula, información académica, bienestar, relaciones internacionales, emprendimiento |
| 🏅 **Egresados** | Cultura, centro deportivo, biblioteca, trámites de egresados |
| 💼 **Centro de Empleabilidad** | UPT Laboral, prácticas, servicios para alumnos y graduados |
| 🔒 **Intranet** | Mi UPT, aula virtual, correos institucionales |

## 💻 Cómo ejecutar localmente

1. Clona el repositorio:

```sh
git clone https://github.com/nkmelndz/upt-home-page.git
cd upt-home-page
```

2. Abre `index.html` en tu navegador:

```sh
# En Linux / macOS
open index.html

# O simplemente arrastra el archivo al navegador
```

No se requieren dependencias adicionales. El proyecto es puramente estático.

## 🚀 Despliegue

El sitio se despliega automáticamente en GitHub Pages mediante GitHub Actions al hacer push a la rama `main`. El workflow también actualiza la descripción y URL del repositorio automáticamente.
